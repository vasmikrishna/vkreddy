# VK Reddy Web Services — vkreddy.com

Premium-grade digital engineering services delivering highly optimized, fast, and gorgeous web and mobile systems designed to survive high traffic volumes and scale smoothly.

Directed by Principal Architect **Vamsi Krishna Chinipireddy**.

---

## 💎 Key Deliverables Implemented

1. **Obsidian Visual Identity (`style.css`):**
   - High-fidelity dark-glassmorphism visuals utilizing fluid HSL tokens.
   - Elegant custom typography combinations (`Outfit` and `Inter` via Google Fonts).
   - Radial teal and purple glows, backdrop saturation filters, and interactive animations.

2. **Marketing Landing Page (`index.html`):**
   - Interactive 3D layered floating hero dashboard.
   - Comprehensive showcase of core architectural offerings.
   - Highly dynamic, filterable template showcasing 5 beautiful layout presets (SaaS, E-commerce, Portfolio, Corporate, Brochure).
   - Interactive, accessible project contact form with visual loading spinner and animated floating label states.

3. **Standalone Biography Page (`about.html`):**
   - Premium founder narrative detailing execution philosophies, visual transparency, and core tenets.
   - High-contrast visual crop of principal architect Vamsi Krishna Chinipireddy with glassmorphic name overlay card.
   - Secure CTA links to connect directly on LinkedIn.

4. **Client-Side Dynamics (`app.js`):**
   - Custom IntersectionObserver scroll-spy and reveal classes.
   - Real-time client-side templates showcase filtering.
   - Interactive mouse coordinates observer generating custom colored cursor glows over services.

## 🚀 Local Deployment

To spin up a lightweight local server and experience the high-fidelity aesthetics:

```bash
# Using Python 3
python3 -m http.server 8000
```

Then visit:
- **Home Landing:** [http://localhost:8000/index.html](http://localhost:8000/index.html)
- **Biography:** [http://localhost:8000/about.html](http://localhost:8000/about.html)
